# KB 002: Project Activation (Welcome)
**Category:** AI & Cloud Computing (Ursa Major)
**Audience:** BearHelp Tier 1 Support

**Trigger:** Systems Team confirms a new GCP Project (`ucr-ursa-major-...`) is provisioned.

**Action:** Send the "Welcome" email to the PI and designated researchers.

**Template: Welcome / Access Handoff**
> **Subject:** Welcome to Ursa Major: Project [Project ID] is Ready
>
> Hello [PI Name],
>
> Your Google Cloud research environment is live.
>
> **1. Access Your Project:**
> *   **Console:** [https://console.cloud.google.com/home/dashboard?project=[Project ID]](https://console.cloud.google.com/home/dashboard?project=[Project ID])
> *   **IAM/Users:** [https://console.cloud.google.com/iam-admin/iam?project=[Project ID]](https://console.cloud.google.com/iam-admin/iam?project=[Project ID])
>
> **2. AI & Gemini API Setup:**
> *   To use Gemini models programmatically, create an API Key here: [https://console.cloud.google.com/apis/credentials?project=[Project ID]](https://console.cloud.google.com/apis/credentials?project=[Project ID])
> *   *Note:* Treat this key like a password. Do not share it.
>
> **3. Support:**
> *   For billing questions or quota increases, reply to this ticket.
>
> Regards,
> Research Computing
