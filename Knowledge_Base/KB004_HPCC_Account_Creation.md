# KB 004: Account Creation / Access
**Category:** High Performance Computing (HPCC)
**Audience:** BearHelp Tier 1 Support

**Trigger:** User asks "How do I get an HPCC account?"
**Resolution:**
1.  **Direct to Portal:** [https://portal.hpcc.ucr.edu/](https://portal.hpcc.ucr.edu/)
2.  **Instruction:** "Log in with your UCR NetID. If you are a student, you must select your PI/Faculty Advisor from the list to request sponsorship."
3.  **Note:** Account creation is automated but requires PI approval via email.
