CephRDS: Your Data, Amplified.

Secure, scalable, and high-performance object storage designed specifically to accelerate the UCR research data lifecycle. Whether you are transferring terabytes of genomic data or querying a massive database, CephRDS provides the resilient foundation you need.

Core Infrastructure

Unified Object Storage for Modern Science

Funded by the NSF CC* program, CephRDS eliminates data silos. It provides a massive, shared data fabric that connects your laboratory instruments directly to the High-Performance Computing Center (HPCC) and the national grid.

S3

100% S3 API Compatible

Write your code once. If your data pipelines use AWS S3 standard libraries like boto3, they will work flawlessly with CephRDS by simply changing the endpoint URL.

Extreme Throughput

Directly connected to the UCR Science DMZ via a 100Gbps network backbone, ensuring your data moves as fast as your computations require.

Unmatched Data Durability

Utilizing an advanced 6+3 Erasure Coding profile, your data is striped across multiple physical drives and nodes. We can sustain multiple simultaneous hardware failures with zero data loss.

Secure Access Control

Maintain strict, granular control over your lab's data. External collaborators are granted scoped, API-level access manually approved by the PI, completely bypassing complex external federation portals.
